# Packages

Reusable public-safe helpers live here.

Possible packages:

- `calm-ui`: generic unbranded UI primitives.
- `privacy-utils`: mock-data helpers, local export helpers, redaction helpers.
- `byo-ai-client`: tiny examples for user-provided API keys.

Rules:

- Keep packages small and readable.
- Use placeholder values for keys and endpoints.
- Do not include production Quietly Tools internals.
- Do not include auth, payments, analytics, private backend logic, or production schemas.
- Every package should document its public safety boundary.
