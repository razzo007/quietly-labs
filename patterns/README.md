# Patterns

Generic calm UI and interaction examples live here.

Good candidates:

- Calm card.
- Breathing orb.
- Ritual ring.
- Empty state.
- Local import/export panel.
- Review-before-apply table.
- Gentle error state.

Rules:

- Teach the idea without copying production Quietly Tools implementation.
- Avoid premium brand assets, logos, monk silhouettes, production orb visuals, and exact layouts.
- Prefer accessible HTML, clear labels, and small examples.
- Use mock data only.
